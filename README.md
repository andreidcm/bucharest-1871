## Use

```bash
git clone git@github.com:andreidcm/bucharest-1871.git && \
  cd redux-all-is-list && \
  npm run setup

# run tests (any `*.test.js`) once
npm test

# watch `src` folder for changes and run test automatically
npm run tdd
```
